# Python Practical Questions – Functions & Parameters

---

## 1. Simple Function
Write a program that:
- Creates a function `show_message()`  
- Prints `"Welcome to Python Functions"`  
- Calls the function 3 times  

---

## 2. Function for Addition
Write a program that:
- Creates a function `add_values(a, b)`  
- Prints sum of two numbers  
- Calls function using different values  

---

## 3. Function with User Input
Write a program that:
- Takes two numbers as input  
- Sends them to a function  
- Function prints multiplication result  

---

## 4. String Concatenation Function
Write a program that:
- Creates function `full_name(first, last)`  
- Prints full name by joining both strings  

---

## 5. Default Parameters
Write a program that:
- Creates function `multiply_numbers(x=1, y=1)`  
- Prints multiplication result  
- Calls function:
  - Without arguments  
  - With one argument  
  - With two arguments  

---

## 6. Keyword Arguments
Write a program that:
- Creates function `student_info(name, age)`  
- Prints both values  
- Calls function using keyword arguments in different order  

---

## 7. Function Returning Value
Write a program that:
- Creates function `square(num)`  
- Returns square of number  
- Stores returned value in variable and prints it  

---

## 8. Return Multiple Values
Write a program that:
- Creates function `calculate(a, b)`  
- Returns:
  - Addition  
  - Subtraction  
- Prints both returned values  

---

## 9. Function with Default and Keyword Arguments
Write a program that:
- Creates function `display_product(product="Keyboard", price=500)`  
- Calls function:
  - Without arguments  
  - With one keyword argument  
  - With both keyword arguments  

---

## 10. Combined Functions Practice
Write a program that:
- Creates function `user_details(name, city, age=18)`  
- Prints all details clearly  
- Calls function:
  - Using positional arguments  
  - Using keyword arguments  
  - Using default age value
